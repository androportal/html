GETTING STARTED WITH PYTHON


Executing Your first Python programme

Select example from option -> get output by pressing execute button

Saving Your Code file

Select save from option -> enter name of file in the pop up dialog -> Press Ok to save in /sdcard/APL/python/code

Opening a Saved file

Select open button -> select the file desired file


Make sure that you have sdcard mounted on device before you can use save and open feature
