Aakash Programming Lab is web based programming console consisting of programming languages like C, C++, Python and Scilab. This is the first time that all these programming languages are included in one application on Android platform. The major attraction is Scilab, which is free and open source software for Nummerical computation. APL uses a chroot environment inside an Andriod environment.
